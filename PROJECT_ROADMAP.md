# Windows ERP Project Roadmap

## Vision
Build a modern offline-first Windows ERP using .NET 8, WPF, MVVM, and SQL Server Express.

## Phase 1
- Solution structure
- Authentication
- Dashboard
- Database layer

## Phase 2
- Inventory
- Purchasing
- Sales
- Customers
- Suppliers

## Phase 3
- Manufacturing
- HR
- Payroll
- Accounting
- Reporting

## Phase 4
- Backup/Restore
- Settings
- Cloud sync preparation

Each phase will be committed incrementally and tested before moving to the next.
